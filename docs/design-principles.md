# Design Principles

Ekiek should feel like a native personal operating layer, not an AI SaaS template.

- Quiet, precise, private, local.
- Agent beside the work, not chatbot in a box.
- Normal files are first-class.
- Context is compiled, not dumped.
- Memory is visible and approved.
- Every source and exclusion is inspectable.
- Browser content is untrusted data.
- Sensitive actions are draft or propose by default.

The interface uses product surfaces, citations, packet chips, and approval states as the visual language.
