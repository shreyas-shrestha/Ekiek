# Future Tauri Daemon

A production Ekiek desktop app can use:

- Tauri desktop shell.
- Rust or Node local daemon.
- SQLite or SQLCipher local store.
- FTS and optional local vector index.
- File watchers.
- Spreadsheet, PDF, Office document, and markdown parsers.
- Codebase parser using tree-sitter or LSP.
- Local model router.
- MCP server.
- Local REST API.
- Browser extension bridge.
- IDE extension bridge.
- Permission broker.
- Action audit log.
- No hosted memory.
- No server-side embeddings.
- Optional user-controlled sync later.
- Optional cloud inference only with explicit consent.

The web prototype models these boundaries with typed mock data and a functioning context compiler.
